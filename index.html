Diploma in Information Technology Applications (CDTA)

The CDTA (Certificate in Digital Technology Applications) course is a comprehensive training program designed to equip students with the foundational and intermediate skills necessary to thrive in the software and application development field.

Course Highlights

The DDTA course includes a blend of programming languages, web technologies, and fundamental computing concepts:

1. Fundametal

  • Variables, data types, and operators
  • Control structures (if, switch, loops)
  • Functions and arrays
  • Pointers and structures
  • File handling in C

2. Windows

  • HTML structure and tags
  • Forms, tables, and lists
  • Embedding images, videos, and links
  • Semantic HTML5 elements

3. Ms Word

  • Java syntax and structure
  • OOP principles (classes, objects, inheritance, polymorphism)
  • Exception handling and file I/O
  • GUI development with Swing

4. MS Excel

  • Programming
  • Form Designing
  • Data Base Connectivity

5. MS PowerPoint

  • Programming
  • Form Designing
  • Data Base Connectivity

6. MS Access

  • Programming
  • Form Designing
  • Data Base Connectivity

4. Cyber Security

  • Programming
  • Form Designing
  • Data Base Connectivity

4. Mobile Operating System

  • Programming
  • Form Designing
  • Data Base Connectivity
